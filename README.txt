READ ME

Description: The program counts and finds inversion sequences avoiding
consecutive patterns of length 4.

Instructions: The main functions are in the capitalized files. The remaining
files are auxiliary functions, with the exception of test4.py and results.txt. The file test4.py runs examples of all the main functions. The file results.txt contains the result of computations in test4.py for the first few values.    

Built with: Python 2.7.12

Author: Juan Auli
